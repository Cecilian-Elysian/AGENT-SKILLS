# AGENT-SKILLS

AI Agent 技能模板库 - 用于 OpenCode 工具的技能定义和模板集合。

## 项目简介

本仓库包含一组可复用的 AI Agent 技能模板，用于在 OpenCode 中执行特定任务。每个技能（Skill）定义了清晰的工作流程、输入输出格式和执行规范。

## 已实现的技能

### obsidian-canvas-creator

**描述**: 在 Obsidian 中创建 `.canvas` 文件，将笔记内容转换为可视化画布。

**功能**:
- 读取 wiki 文件并分析内容结构
- 根据内容类型选择布局策略（中心辐射/流程/分组/自由）
- 生成符合 Obsidian Canvas 格式的 JSON 文件
- 支持 text/image/link 三种节点类型

**使用场景**:
- 用户要求创建 canvas、制作 canvas、生成 canvas
- 请求将笔记/文件内容转换为 canvas 格式
- 使用 `/canvas` 命令

**文件结构**:
```
obsidian-canvas-creator/
├── SKILL.md                    # 技能定义
└── references/
    └── canvas-format.md        # Canvas JSON 格式参考
```
